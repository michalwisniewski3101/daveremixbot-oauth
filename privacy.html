Privacy Policy

DaveRemixBot stores uploaded media files, draft metadata, publishing logs, and protected OAuth token data in the private local application instance controlled by the operator.

  • This public site does not store application secrets.
  • This public site does not contain the TikTok client secret, refresh tokens, or Gemini keys.
  • OAuth authorization codes are forwarded to the private local callback for token exchange.
  • External platform APIs are called only by the private local application after operator authorization.

The public OAuth bridge exists only to provide a static HTTPS redirect URI required by supported platform OAuth flows.
